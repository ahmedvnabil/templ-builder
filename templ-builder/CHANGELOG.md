# Changelog

All notable changes to the **Templ Builder** project will be documented in this file.

## [0.1.0] - 2026-06-15
### Added
- Initial Release.
- Custom Post Type `templ_item` and Custom Taxonomies (`templ_type`, `templ_collection`).
- Centralized Structured Meta Fields configuration (Priority, Badge, Icon, Eyebrow, CTA URLs, Custom JSON Fields).
- Elegant Admin Meta Box Interface with section tabs and inline JSON validation.
- Template Manager with 13 pre-defined frontend layout keys (cards, accordion FAQs, testimonials, grids, lists).
- Frontend Renderer that normalizes CPT metadata safely and renders layouts.
- Settings Page under `Settings > Templ Builder` for configuring defaults, shadows, corners, and custom CSS.
- Tools Page under `Tools > Templ Builder` with a Live Shortcode Builder, JSON Import/Export backup tools, and Demo Content Seeder.
- Static HTML snippet exporter.
- Responsive CSS Stylesheet with RTL support and keyboard navigation focus rings.
