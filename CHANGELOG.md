# Changelog

All notable changes to the **Templ Builder** WordPress plugin will be documented in this file.

This project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [0.1.0] - 2026-06-15

### Added
- **CPT templ_item**: Custom post registry to save template card parameters (eyebrow, subtitle, button_label, secondary_url, price, rating, role_label, organization, location, and accent colors).
- **13 Visual Card Previews**: Layout structures supporting cards, grid, list, section, feature, testimonial, faq, portfolio-card, app-card, service-card, social-proof, minimal, and single showcase.
- **Custom JSON Fields**: Flexible key-value structures enqueued in a syntax-checked, formatting-enabled JSON editor in the metabox.
- **Demo Content Seeder**: One-click mock post generator to inspect all 9 use cases instantly without duplication.
- **JSON Portability Backup Systems**: Export and import Templ Items in a single JSON file. Bypasses duplicate imports and enforces capability checks.
- **Static HTML Snippet Exporter**: Download responsive, escaped, raw HTML block snippets to copy-paste into static pages.
- **RTL & Arabic Optimization**: Native support for Arabic typography, metadata adjustments, and right-to-left layouts.
- **Accessibility (a11y) Integrated**: Fully keyboard navigable (`:focus-visible`), aria-labeled, screen-reader friendly (`.tb-sr-only`), and compliant color contrasts.
- **Security-Reviewed**: Strict nonces, capability validation, draft-by-default imports, and Custom CSS HTML-tag stripping sanitizers.
